# MapPlot 打包說明

## 已完成打包

執行檔已成功建立在 `dist\MapPlot.exe`

### 執行檔資訊
- **檔案名稱**: MapPlot.exe
- **檔案大小**: 約 11 MB
- **類型**: Windows 可執行檔 (.exe)
- **功能**: 地圖繪製工具,包含完整的 GUI 介面

## 如何使用

1. 直接雙擊 `dist\MapPlot.exe` 即可執行
2. 執行檔已經包含所有必要的依賴套件,無需安裝 Python 環境
3. 首次執行時可能需要較長的啟動時間

## 重新打包

如果修改了程式碼並需要重新打包,有兩種方式:

### 方式 1: 使用 PowerShell 腳本 (推薦)
```powershell
.\build_exe.ps1
```

### 方式 2: 手動執行命令
```powershell
.\.venv\Scripts\pyinstaller.exe --name MapPlot --onefile --noconsole main.py
```

## 打包選項說明

- `--name MapPlot`: 指定執行檔名稱為 MapPlot.exe
- `--onefile`: 打包成單一執行檔
- `--noconsole`: 不顯示控制台視窗 (GUI 應用程式)

## 注意事項

1. **檔案大小**: 執行檔包含了 Python 解譯器和所有依賴套件,所以檔案較大是正常現象
2. **防毒軟體**: 某些防毒軟體可能會誤判,需要加入信任清單
3. **配置檔**: `config.json` 會在執行檔第一次執行時自動建立
4. **啟動時間**: 首次啟動可能需要 5-10 秒,這是正常現象

## 依賴套件

打包時已包含以下主要套件:

## 疑難排解

### 執行檔無法啟動
1. 確認 Windows 版本為 Windows 10 或更新版本
2. 檢查是否被防毒軟體阻擋
3. 以系統管理員身分執行

### 重新打包失敗
1. 確認虛擬環境已啟動: `.\.venv\Scripts\Activate.ps1`
2. 確認 PyInstaller 已安裝: `pip list | Select-String pyinstaller`
3. 清理 build 和 dist 目錄後重試
