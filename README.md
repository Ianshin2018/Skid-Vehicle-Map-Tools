## 1. 程式啟動

執行主程式：

```
python main.py
```

程式啟動後會在背景自動預載各樓層地圖資料，視窗右下方的進度條會顯示載入進度。預載完成後樓層按鈕才會啟用。

> **注意**：預載期間樓層按鈕呈灰色不可點擊，請等待進度條完成後再操作。

## 3. 功能列

功能列位於視窗頂端，包含以下三個選單：

### 檔案

| 項目 | 說明 |
|------|------|
| 匯入資料集 | 匯入外部 CSV 資料集，合併至現有打滑記錄 |
| 匯出圖片 | 將目前畫布畫面（含圖例）另存為 PNG / JPG |

| 項目 | 說明 |

| ☐ 打滑位置 | 勾選後在地圖上疊加顯示打滑方框（需先選擇日期） |

### 說明

| 項目 | 說明 |
|------|------|
| 使用說明書 | 在瀏覽器中開啟本說明書 |

> **提示**：「顯示」選單的勾選狀態與選項列的同名 checkbox 互相同步，操作效果相同。

## 4. 樓層選擇

地圖畫布**左上角**有三個樓層按鈕：**1F**、**2F**、**3F**。

1. 點擊任一樓層按鈕即可載入該樓層地圖。
2. 若背景預載已完成（進度條跑完），切換速度非常快；尚未預載則需等待數秒。
3. 切換樓層後，日期列表與排名表格會同步清空，需重新選擇日期。

| 樓層 | 車輛範圍 |
|------|----------|
| 1F | 101 ~ 115 |
| 2F | 116 ~ 137、140 |
| 3F | 138、139 |

## 5. 日期篩選

位於**左側面板**，用於篩選要分析的打滑資料日期。

### 操作步驟

1. 點選樓層按鈕後，日期列表會自動出現，以週為單位顯示（例如：`W03  1/14~1/20`）。
2. 勾選想要分析的日期（支援多選）。
3. 頂端快速按鈕：
   - **全選**：勾選所有日期
   - **清除**：取消所有日期勾選
4. 選擇日期後，地圖打滑方框與右側排名表格會自動更新。

> **注意**：若選項列的「打滑位置」未勾選，打滑方框不會出現在地圖上，但排名仍會更新。

## 6. 圖層顯示控制

地圖由三個圖層疊加組成：

| 圖層順序 | 名稱 | 顏色說明 | 預設狀態 |
|----------|------|----------|----------|
| 第一層（底圖） | 車輛路線圖 | 藍色箭頭表示行駛方向 | 固定顯示 |
| 第二層 | 施工區域 | 水藍色半透明方塊 | **預設關閉** |
| 第三層 | 打滑方框位置 | 紅色外框方塊 | **預設關閉** |

> **注意**：程式啟動時施工區域與打滑位置圖層預設為關閉，需手動勾選才會顯示。

### 切換方式（以下兩種效果相同）

- **功能列 → 顯示 → 施工區域 / 打滑位置**
- 滑桿與畫布之間的選項列：**☐ 施工區域 / ☐ 打滑位置** checkbox

## 7. 地圖瀏覽操作

### 縮放

| 操作 | 說明 |
|------|------|
| `Ctrl` + 滾輪向上 | 以滑鼠游標為中心**放大** |
| `Ctrl` + 滾輪向下 | 以滑鼠游標為中心**縮小** |
| 畫布右上角 **+** 按鈕 | 固定比例放大（退出全圖模式） |
| 畫布右上角 **-** 按鈕 | 固定比例縮小（退出全圖模式） |

> 單獨滾動滾輪（不按 Ctrl）為垂直捲動地圖視野。

### 平移

在地圖上**按住左鍵並拖曳**即可平移視野。

### 捲動條

畫布右側與下方的捲動條可移動地圖視野。

## 8. 全圖模式

將地圖縮放至完整顯示於畫布內，方便一次瀏覽整個樓層地圖。

### 啟用 / 關閉方式

| 操作 | 說明 |
|------|------|
| 勾選選項列的 **☐ 全圖** checkbox | 進入全圖模式 |
| 再次點擊 **☐ 全圖**（取消勾選） | 還原至進入全圖前的縮放比例與捲動位置 |
| 按 **+** / **-** 或使用 `Ctrl` + 滾輪縮放 | 退出全圖模式，進入一般縮放狀態 |

### 全圖模式特色

- 圖片會置中顯示，畫布背景變為**淺灰色**。
- 視窗尺寸調整時，縮放比例會自動跟著重新計算，始終保持全圖顯示。

----

## 9. 放大鏡功能

在畫布右下角顯示一個即時放大方框，方便查看地圖局部細節。

### 啟用方式

勾選選項列的 **☐ 放大鏡** checkbox。

### 使用方式

啟用後，移動滑鼠游標至地圖上，右下角方框會即時顯示游標附近的放大畫面。

| 項目 | 說明 |
|------|------|
| 方框尺寸 | 120 × 120 像素 |
| 放大倍率 | 約 5 倍（從原圖取樣） |
| 防抖延遲 | 5 ms（滑鼠移動 < 5 像素時自動跳過更新） |

### 顯示內容

放大鏡方框會同時顯示以下所有圖層內容：

| 圖層 | 是否顯示 |
|------|----------|
| 底圖（路線圖） | ✓ 顯示 |
| 施工區域 | ✓ 顯示（若已開啟） |
| 打滑方框（紅框） | ✓ 顯示（若已選日期） |
| 車輛位置標記（綠框） | ✓ 顯示（若已點選車輛） |

> 若要關閉放大鏡，再次點擊 **☐ 放大鏡** 取消勾選即可。

## 10. 重複打滑門檻滑桿

選擇日期後，畫布上方出現滑桿，可過濾「重複打滑次數」不足的站點。

### 滑桿對照表

| 滑桿位置 | 顯示條件 |
|----------|----------|
| 最左端（0） | 顯示所有出現過打滑的站點（最少 1 次） |
| 中間（50，預設） | 顯示次數 ≥ 中間值的站點 |
| 最右端（100） | 只顯示次數最高的站點 |

滑桿標籤即時顯示：**目前門檻次數 / 中間值 / 最高次數**

移動滑桿後，地圖方框與右側排名表格會在約 **150 ms** 後自動更新。

----

## 11. 打滑次數排名

位於**右側面板**上方，列出各車輛在所選日期內的打滑次數排名。

| 欄位 | 說明 |
|------|------|
| 名次 | 依打滑次數由高到低排列 |
| 車號 | 車輛編號 |
| 次數 | 所選日期內的總打滑次數 |

### 在地圖上標示特定車輛

1. 確認右側面板頂端的 **☐ 顯示車輛位置** 已勾選。
2. 點擊排名表格中的任一車號。
3. 該車輛的打滑站點以**綠色粗外框**在地圖上高亮顯示。
4. 啟用放大鏡時，放大方框內也會同步顯示車輛位置綠框。

> 取消勾選 **☐ 顯示車輛位置** 即可清除地圖上的綠框標記。

## 12. 站點資訊查詢

在地圖上對打滑方框按下**右鍵**，可查看該站點的詳細資訊。

### 顯示內容

| 方框類型 | 顯示資訊 |
|----------|----------|
| 地址站點（Address） | Address ID |
| 路段站點（Section） | Section ID、起點地址（From）、終點地址（To） |

資訊會以浮動小視窗顯示在點擊位置旁，**4 秒後自動關閉**。

## 13. 匯出地圖圖片

將目前畫布可見的所有圖層儲存為圖片檔，並自動附加圖例。

### 操作方式

- **功能列 → 檔案 → 匯出圖片**
- 畫布下方的 **匯出圖片** 按鈕

### 步驟

1. 點擊上述任一入口。
2. 在彈出視窗中選擇**儲存位置**與**檔案名稱**。
3. 選擇格式：**PNG**（建議）或 **JPG**。
4. 點擊確認即完成儲存。

### 圖片內容

匯出圖片會在地圖右側自動附加圖例：

| 圖例項目 | 說明 |
|----------|------|
| → 藍色箭頭 | 車輛行駛方向 |
| ■ 水藍色方塊 | 施工區域（僅在勾選時出現） |
| □ 紅色外框 | 打滑方框位置（僅在勾選時出現） |
| ! 紅色 | 異常地址 |
| ! 橘色 | 異常路段 |

## 14. 匯入資料集

將外部 CSV 格式的打滑記錄合併至現有資料，無需手動編輯 `highlights.csv`。

### 操作方式

**功能列 → 檔案 → 匯入資料集**

彈出檔案選擇視窗後，選取目標 CSV 檔案即可。

### CSV 格式需求

匯入的 CSV 檔案必須包含以下四個必要欄位（大小寫不敏感）：

| 欄位名稱 | 說明 |
|----------|------|
| `start_date` | 打滑發生日期（例如：`2025-01-15`） |
| `number` | 車輛編號（例如：`105`） |
| `addressid` | 打滑發生的地址站點 ID |
| `sectionid` | 打滑發生的路段站點 ID |

> `addressid` 與 `sectionid` 可擇一填寫；兩者皆填亦可。

### 驗證流程

匯入時依序執行以下四道驗證：

| 步驟 | 說明 | 未通過時的處理 |
|------|------|----------------|
| 1. 欄位檢查 | 確認四個必要欄位均存在 | 終止匯入，顯示缺少的欄位名稱 |
| 2. 格式驗證 | `start_date` 與 `number` 不可為空值 | 該列排除，不匯入 |
| 3. 地圖驗證 | `addressid` 或 `sectionid` 至少有一個能對應各樓層地圖資料 | 兩者皆不符者排除，不匯入，並在結果視窗顯示異常明細 |
| 4. 重複比對 | 與現有記錄比對四個必要欄位，完全相同的列視為重複 | 重複列略過，不重複寫入 |

### 匯入結果通知

完成後會顯示詳細結果視窗，包含以下資訊：

```
匯入來源：my_data.csv
資料總筆數：120 筆
成功匯入：85 筆
重複略過：10 筆
格式異常：3 筆
地圖不符：7 筆

備註：
・有 3 筆缺少 start_date 或 number 欄位值
・有 7 筆的 addressid 與 sectionid 均無法對應各樓層地圖資料，已排除（不匯入）
・  異常明細（前5筆）：
・  日期=2025-01-15  車號=105  addressid=999999  sectionid=99999
```

| 欄位 | 說明 |
|------|------|
| 資料總筆數 | 匯入 CSV 的原始列數 |
| 成功匯入 | 通過全部驗證並新增至記錄的筆數 |
| 重複略過 | 與現有資料完全相同而跳過的筆數 |
| 格式異常 | start_date 或 number 為空而排除的筆數 |
| 地圖不符 | addressid 與 sectionid 均無法對應地圖資料而排除的筆數 |

### 注意事項

- 成功匯入的資料會**立即存回** `highlights.csv` 並更新記憶體，無需重新啟動程式。
- 若目前已載入樓層，匯入成功後日期面板與打滑排名會**自動刷新**。
- 地圖不符的異常資料**不會被匯入**，但結果視窗會列出前 5 筆明細供確認。

## 15. 警報與異常資訊

### 右側異常資訊區（紅底）

顯示資料讀取或驗證過程中發現的問題，例如：
- 座標資料異常
- 找不到打滑日誌檔案
- 路段或地址 ID 無法對應

### 狀態列（視窗頂部）

即時顯示目前操作狀態：

| 狀態訊息範例 | 說明 |
|-------------|------|
| `就緒 - 1F` | 1F 地圖已載入完成 |
| `載入樓層 2F（使用快取）` | 正在讀取已快取的資料 |
| `更新打滑方框 (3 個日期)...` | 正在依所選日期重繪地圖 |
| `圖層: 施工區域[顯示] 打滑位置[隱藏]` | 切換圖層後的狀態回報 |

## 16. 常見問題 / 排除辦法

---

### ▌ 程式啟動 / 載入問題

#### 程式啟動後樓層按鈕無法點擊

**原因**：背景預載尚未完成。
**排除**：等待視窗右下方進度條跑完，按鈕即自動啟用，無需手動重試。

#### 程式啟動速度很慢

**原因**：程式啟動時會在背景同時預載三個樓層的地圖資料，資料量大時需要較多時間。
**排除**：進度條完成前可保持程式開啟，不需關閉重開。若明顯超過 2 分鐘仍未完成，請確認 `Map/` 資料夾完整存在。

#### 點選樓層後地圖沒有顯示，畫布維持空白

**原因**：可能是地圖資料夾缺失、路徑錯誤或資料格式問題。
**排除步驟**：
1. 確認 `Map/` 資料夾下存在 `Garmin1F`、`Garmin2F`、`Garmin3F` 子資料夾。
2. 確認每個子資料夾內有 `Address.csv` 與 `Section.csv`。
3. 查看右側**異常資訊區（紅底）**是否有具體錯誤訊息。
4. 查看狀態列是否顯示錯誤說明。

#### 切換樓層後地圖顯示舊樓層的內容

**原因**：快取資料的正常行為，切換動作已觸發但畫布更新中。
**排除**：此為正常短暫過渡，狀態列顯示「就緒 - XF」即代表載入完成。若長時間未更新，嘗試再次點擊樓層按鈕。

---

### ▌ 地圖顯示問題

#### 地圖上看不到施工區域（水藍色方塊）

**原因**：「施工區域」圖層預設為關閉。
**排除**：勾選選項列的 **☐ 施工區域** 或功能列 **顯示 → 施工區域**。若勾選後仍無顯示，確認當前樓層資料夾內有 `zone.csv` 或 `zoneSection.csv`。

#### 選了日期但地圖上沒有出現打滑方框（紅框）

**原因**：可能有以下幾種：
**排除步驟**：
1. 確認選項列的 **☐ 打滑位置** 已勾選（預設為關閉）。
2. 確認**打滑門檻滑桿**未調至最右端（會過濾掉所有次數不夠高的站點）。
3. 確認 `highlights.csv` 存在且有所選日期的資料。
4. 確認所選日期與資料中的 `start_date` 格式一致（例如：`2025-01-15`）。

#### 打滑方框位置看起來偏移，與路線圖不對齊

**原因**：可能是切換樓層後底圖尺寸與 overlay 不一致。
**排除**：重新點選同一樓層按鈕重新載入地圖，讓程式重新對齊圖層。

#### 地圖上找不到車輛位置綠框

**原因**：可能是車輛勾選未啟用，或尚未選取日期。
**排除步驟**：
1. 確認右側面板頂端的 **☐ 顯示車輛位置** 已勾選。
2. 確認已先選取日期（無日期則無打滑記錄，排名表格為空，無從點選）。
3. 在排名表格中**點擊車號列**觸發高亮，僅瀏覽排名不會自動顯示綠框。

---

### ▌ 日期與排名問題

#### 切換樓層後日期列表消失，排名表格是空的

**說明**：此為正常行為。每次切換樓層，日期列表與排名會清空，需重新勾選日期後才會更新。

#### 日期列表沒有資料（點選樓層後列表空白）

**原因**：`highlights.csv` 不存在或其中無與當前樓層車號對應的記錄。
**排除步驟**：
1. 確認 `highlights.csv` 位於程式根目錄（與 `main.py` 同層）。
2. 確認 CSV 中的 `number` 欄位包含對應樓層的車號範圍（1F: 101–115；2F: 116–137,140；3F: 138,139）。
3. 確認 `start_date` 欄位格式正確（例如：`2025-01-15`），避免空值或格式異常。

#### 打滑門檻滑桿移動後地圖沒有反應

**原因**：滑桿有約 150 ms 防抖延遲，快速移動後需稍等片刻。
**排除**：停止移動後等待一下，地圖與排名會自動更新。若仍無反應，確認已有日期被勾選。

---

### ▌ 全圖 / 縮放問題

#### 勾選「全圖」後畫面沒有變化

**原因**：地圖尚未載入（尚未點選樓層）。
**排除**：請先點選任一樓層按鈕等地圖顯示後，再使用全圖功能。

#### 取消「全圖」勾選後視野沒有還原到之前位置

**原因**：若在全圖模式下曾使用 **+** / **-** 按鈕或 `Ctrl` + 滾輪縮放，全圖模式已自動退出，此時「還原」功能不保留舊位置。
**排除**：重新進入全圖模式（再次勾選 **☐ 全圖**），然後取消勾選可還原至進入前的狀態。

#### Ctrl + 滾輪縮放沒有反應

**原因**：滑鼠游標需移至地圖畫布上才能觸發縮放，游標在畫布外（如日期列表或排名區）時不會縮放。
**排除**：將游標移至地圖畫布內後再操作。

---

### ▌ 放大鏡問題

#### 啟用放大鏡後方框顯示空白或灰色

**原因**：地圖尚未載入完成。
**排除**：先點選樓層按鈕，等地圖正常顯示後再啟用放大鏡。

#### 放大鏡內看不到打滑方框或施工區域

**原因**：放大鏡顯示的是合成後的圖層，若打滑位置或施工區域圖層為關閉狀態，放大鏡內也不會顯示。
**排除**：先勾選 **☐ 施工區域** 和 **☐ 打滑位置** 並選取日期，再使用放大鏡。

#### 放大鏡內看不到車輛位置綠框

**原因**：車輛綠框需先在排名表格中點擊車號觸發才會出現。
**排除**：確認 **☐ 顯示車輛位置** 已勾選，並在排名表格中點擊目標車號後，放大鏡即會同步顯示綠框。

---

### ▌ 匯出圖片問題

#### 匯出的圖片是純灰色或空白

**原因**：地圖尚未載入或圖層皆為關閉狀態。
**排除**：
1. 先點選樓層並等待地圖顯示完成。
2. 若需包含打滑方框或施工區域，請先勾選相應圖層並選取日期後再匯出。

#### 匯出圖片沒有圖例

**原因**：僅勾選了底圖，未開啟任何附加圖層。
**排除**：圖例項目會根據勾選的圖層動態顯示，至少需開啟施工區域或打滑位置其中一層。

---

### ▌ 匯入資料集問題

#### 匯入時出現「缺少必要欄位」錯誤

**原因**：CSV 欄位名稱有誤或缺少必要欄位。
**排除**：確認 CSV 檔案包含以下四個欄位（大小寫不敏感）：`start_date`、`number`、`addressid`、`sectionid`。若欄位存在但名稱不同，請手動重新命名後再匯入。

#### 匯入後顯示「成功匯入 0 筆」

可能原因與排除方式：

| 可能原因 | 排除方式 |
|----------|----------|
| 所有資料均為重複 | 匯入資料已存在於 `highlights.csv`，無需處理 |
| 所有資料均為格式異常 | 確認 `start_date` 與 `number` 欄位值非空 |
| 所有資料均地圖不符 | 確認 `addressid` 或 `sectionid` 的值與地圖資料一致 |

#### 匯入後顯示「地圖不符」

**原因**：該筆資料的 `addressid` 與 `sectionid` 均無法對應至任一樓層（Garmin1F / 2F / 3F）的地圖資料，可能是 ID 數值有誤或來自不同地圖版本。
**排除**：對照 `Map/Garmin{N}F/Address.csv` 與 `Section.csv` 確認正確的 ID 值後重新匯入。

#### 匯入成功後地圖沒有更新

**原因**：匯入當下尚未選取樓層，無法自動刷新。
**排除**：匯入完成後點選樓層按鈕，再重新勾選日期，地圖與排名即會更新。

---

### ▌ 站點資訊查詢問題

#### 右鍵點擊打滑方框後沒有跳出資訊視窗

**原因**：可能點擊位置未精確落在方框範圍內，或目前無打滑方框可點擊。
**排除**：
1. 確認已選取日期且打滑位置圖層已勾選，地圖上有紅框存在。
2. 嘗試放大地圖後再精確點擊方框範圍。
3. 若已在全圖模式，可先縮放至較大比例再嘗試右鍵。

#### 資訊視窗出現後馬上消失

**說明**：資訊視窗設計為 4 秒後自動關閉，此為正常行為。如需再次查看，重新右鍵點擊即可。
